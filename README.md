Customer facing homepage.

## Form submission backend - Formspree

Formspree offers a simple integration for form submission. Free tier includes 50 submissions per month.

To activate it:
* Sign up at formspree.io
* Create a form pointed at `miller.tradgard@gmail.com`
* Replace `XXXXXXXX` in the JS with the form ID they give you

## Github pages deployment

Page hosting with Github pages
* Free
* Soft bandwidth limit is 100 GB/month — site will use maybe a few hundred MB
* Note - requires that project is public

Setup:
* Create a CNAME record with DNS provider referring to github pages site
* Update github pages settings with the custom domain
